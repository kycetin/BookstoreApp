## Getting Started


 ### Recommended Installation With Docker
```bash
 git clone https://github.com/Zrrck/Bookstore-App.git
 cd Bookstore-App/
 sudo docker-compose up
 ```

 -Open one of these;
 - http:localhost:8000
 - http:127.0.0.1:8000
 - http:0.0.0.0:8000

### Without Docker

```bash
 git clone https://github.com/Zrrck/Bookstore-App.git
 cd Bookstore-App/
 pip install -r requirements.txt
 sudo python3 manage.py runserver
 ```
 -Open one of these;
 - http:localhost:8000
 - http:127.0.0.1:8000
 - http:0.0.0.0:8000
